## ADDED Requirements

### Requirement: AI Outline Generation

Teachers SHALL generate outlines using AI.

#### Scenario: Generate outline

- **WHEN** a teacher inputs topic
- **THEN** AI creates outline

### Requirement: Topic Analysis

The system SHALL analyze topics for outlines.

#### Scenario: Analyze topic

- **WHEN** topic is provided
- **THEN** structure is suggested

### Requirement: Outline Customization

Teachers SHALL edit AI-generated outlines.

#### Scenario: Customize outline

- **WHEN** AI generates outline
- **THEN** teacher can modify it
