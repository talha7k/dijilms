## ADDED Requirements

### Requirement: AI Content Generation

Teachers SHALL generate lesson content using AI.

#### Scenario: Generate notes

- **WHEN** a teacher specifies a topic
- **THEN** AI generates study notes

### Requirement: TinyMCE Integration

The system SHALL use TinyMCE for content editing.

#### Scenario: Edit content

- **WHEN** teacher edits generated content
- **THEN** TinyMCE editor is available

### Requirement: Content Saving

Teachers SHALL save generated content.

#### Scenario: Save content

- **WHEN** teacher finishes editing
- **THEN** content is stored and linked to lessons
