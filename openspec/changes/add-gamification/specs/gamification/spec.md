## ADDED Requirements

### Requirement: Points System

The system SHALL award points for activities.

#### Scenario: Earn points

- **WHEN** a student completes a lesson
- **THEN** points are awarded

### Requirement: Leaderboards

The system SHALL display leaderboards.

#### Scenario: View leaderboard

- **WHEN** a user checks rankings
- **THEN** top performers are listed

### Requirement: Achievements

The system SHALL grant achievements.

#### Scenario: Unlock achievement

- **WHEN** criteria are met
- **THEN** an achievement is awarded
