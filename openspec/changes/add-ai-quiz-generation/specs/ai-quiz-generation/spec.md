## ADDED Requirements

### Requirement: AI Quiz Generation

Teachers SHALL generate quizzes using AI.

#### Scenario: Generate quiz

- **WHEN** a teacher inputs lesson content
- **THEN** AI creates questions

### Requirement: Content Analysis

The system SHALL analyze content for quiz creation.

#### Scenario: Analyze lesson

- **WHEN** content is provided
- **THEN** key concepts are identified

### Requirement: Quiz Customization

Teachers SHALL edit AI-generated quizzes.

#### Scenario: Customize quiz

- **WHEN** AI generates a quiz
- **THEN** teacher can modify it
