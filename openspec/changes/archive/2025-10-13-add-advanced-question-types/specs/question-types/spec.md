## ADDED Requirements

### Requirement: Multiple Question Types

The system SHALL support various question formats.

#### Scenario: Create MCQ

- **WHEN** an instructor selects MCQ
- **THEN** the question type is available

### Requirement: Question Creation

Instructors SHALL create questions of different types.

#### Scenario: Add question

- **WHEN** creating an assessment
- **THEN** various types can be chosen

### Requirement: Question Answering

Students SHALL answer different question types.

#### Scenario: Answer matching

- **WHEN** taking a matching question
- **THEN** drag-and-drop works
