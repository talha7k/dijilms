## Why

The home page features section currently highlights Firebase and SvelteKit template capabilities, but as a Learning Management System (LMS), it should showcase the core LMS features to demonstrate value to potential users and educators.

## What Changes

Update marketing pages to showcase LMS features:

- Home page: Replace features section with LMS capabilities (course management, assessments, progress tracking, etc.)
- Features page: Update feature cards and descriptions to highlight LMS functionalities
- Pricing page: Revise plans and features to reflect LMS tiers (e.g., individual educators, schools, enterprises)

## Impact

- Affected code: src/lib/components/marketing/home/features.svelte, src/routes/(marketing)/features/+page.svelte, src/routes/(marketing)/pricing/+page.svelte, src/config.ts
- No breaking changes; content updates to align with LMS purpose.
