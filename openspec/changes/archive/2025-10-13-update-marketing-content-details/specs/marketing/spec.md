## Purpose

The marketing capability handles the presentation and promotion of the LMS platform to potential users.

## ADDED Requirements

### Requirement: Pricing Toggle Visibility

The pricing page SHALL display a visible and functional Monthly/Yearly toggle.

#### Scenario: User toggles billing period

- **WHEN** a visitor views the pricing page
- **THEN** the Monthly/Yearly switch is clearly visible and functional

### Requirement: Remove External Links

The marketing pages SHALL not contain GitHub or other irrelevant external links.

#### Scenario: User navigates marketing pages

- **WHEN** a visitor browses home, features, pricing pages
- **THEN** no GitHub links are present

### Requirement: Updated Contact Information

The contact page SHALL display correct information for Elite Riyadh, KSA.

#### Scenario: User views contact page

- **WHEN** a visitor accesses the contact page
- **THEN** info@enrich.sa and Elite Riyadh details are shown

### Requirement: LMS-Focused Content

All marketing content SHALL be relevant to the LMS platform.

#### Scenario: User views home page

- **WHEN** a visitor accesses the home page
- **THEN** all content promotes LMS features and benefits

### Requirement: Detailed Feature Explanations

The features page SHALL include detailed explanations for each feature.

#### Scenario: User explores features

- **WHEN** a visitor clicks on a feature
- **THEN** detailed information about that feature is displayed
