## ADDED Requirements

### Requirement: Multiple Choice Questions
Teachers SHALL create MCQ questions with single or multiple correct answers.

#### Scenario: Creating MCQ
- **WHEN** teacher selects MCQ type and enters question with options
- **THEN** question is saved with correct answer indicators

### Requirement: Fill in the Blank
Teachers SHALL create FIB questions with text gaps.

#### Scenario: FIB creation
- **WHEN** teacher enters text with blank placeholders
- **THEN** blanks are marked for student input

### Requirement: Short Answer
Teachers SHALL create short answer questions.

#### Scenario: Short answer setup
- **WHEN** teacher defines question and expected keywords
- **THEN** answer validation is configured

### Requirement: Matching
Teachers SHALL create matching questions with pairs.

#### Scenario: Matching pairs
- **WHEN** teacher defines left and right column items
- **THEN** drag-drop interface is available for students

### Requirement: Text Response
Teachers SHALL create essay-type questions.

#### Scenario: Essay questions
- **WHEN** teacher sets word limits and rubric
- **THEN** rich text editor is provided for responses
