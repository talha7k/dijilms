## ADDED Requirements

### Requirement: Forum Creation

Instructors SHALL be able to create discussion forums for courses.

#### Scenario: Instructor creates forum

- **WHEN** an instructor sets up a forum for a course
- **THEN** students can post in it

### Requirement: Post Creation

Users SHALL be able to create posts and replies.

#### Scenario: Student posts message

- **WHEN** a student submits a post
- **THEN** it appears in the forum

### Requirement: Threaded Discussions

The system SHALL support threaded conversations.

#### Scenario: Reply to post

- **WHEN** a user replies to a post
- **THEN** the reply is nested under the original
