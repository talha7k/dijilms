## ADDED Requirements

### Requirement: Badge Creation

Admins SHALL create badges.

#### Scenario: Create badge

- **WHEN** an admin defines a badge
- **THEN** it is available for earning

### Requirement: Badge Earning

Students SHALL earn badges automatically.

#### Scenario: Earn badge

- **WHEN** a student meets criteria
- **THEN** the badge is awarded

### Requirement: Badge Display

Users SHALL view their badges.

#### Scenario: View badges

- **WHEN** checking profile
- **THEN** earned badges are shown
