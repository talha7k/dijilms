## Requirements
### Requirement: Course Browsing

Users SHALL be able to browse available courses.

#### Scenario: User views course catalog

- **WHEN** a user navigates to the courses page
- **THEN** a list of available courses is displayed

### Requirement: Course Enrollment

Users SHALL be able to enroll in courses.

#### Scenario: User enrolls in a course

- **WHEN** a user clicks enroll on a course
- **THEN** they are enrolled and the course appears in their dashboard

### Requirement: Enrollment Management

Users SHALL be able to view and manage their enrollments.

#### Scenario: User views enrolled courses

- **WHEN** a user checks their profile or dashboard
- **THEN** their enrolled courses are listed with options to unenroll
